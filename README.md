# Precision Fire Services - Supabase Registry (PROJECT_FINAL_V12.0)

This application is configured to run on **Supabase**.
**VERSION: PROJECT_FINAL_V12.0**
**UPDATES: Adjusted scoring (1.5% failed, 5% Flash Fire), Fixed Flash Fire Recorded Mass requirement.**

### Environment Setup
Ensure the following variables are configured in your environment:
- `VITE_SUPABASE_URL`: Your Supabase Project URL
- `VITE_SUPABASE_ANON_KEY`: Your Supabase Anonymous Key
- `VITE_GEMINI_API_KEY`: Your Google Gemini API Key

### Deployment
Deployment is handled via standard static web hosting (e.g., Vercel, Netlify, or Supabase Hosting) pointing to the Supabase backend.
